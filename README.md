# AI Soundhelper

Интерактивная система персонализированной генерации музыки на основе нейросетевой модели MusicGen.

## Возможности

- Загрузка до 5 референсных треков для анализа стиля
- Извлечение музыкальных признаков:
  - тональность с определением мажорного/минорного лада
  - темп (BPM)
  - спектральная яркость
  - настроение, плотность ритма, мощность баса, уровень энергии
- Генерация музыки через MusicGen-small
- Сохранение треков и стилевых профилей в библиотеку
- REST API для интеграции с другими приложениями
- Веб-интерфейс с тремя вкладками (Генератор, Профили, Библиотека)

## Быстрый запуск (Windows)

1. Убедитесь, что установлен **Python 3.10+**

2. Скачайте или склонируйте репозиторий

3. Дважды кликните по файлу:
start.bat

text

Скрипт автоматически:
- проверит наличие Python и pip
- установит все необходимые зависимости (FastAPI, PyTorch, transformers, librosa, scipy, numpy)
- создаст нужные папки (uploads, results)
- запустит бэкенд-сервер на `http://localhost:8000`
- откроет веб-интерфейс в браузере

**Важно:** не закрывайте окно с бэкендом, пока работаете с системой.

## Ручной запуск

### Запуск бэкенда

```bash
cd backend
python main.py
```
Сервер запустится на http://localhost:8000
Документация API доступна по адресу http://localhost:8000/docs

Запуск фронтенда
Откройте в браузере файл frontend/index.html

Системные требования
Python 3.10+

-8 ГБ оперативной памяти (рекомендуется)

-Графический ускоритель не обязателен (генерация работает на CPU)

-Поддерживаемые ОС: Windows, macOS, Linux

Технологии
-Python 3.10+ — основной язык

-FastAPI — веб-фреймворк для REST API

-MusicGen-small — генеративная нейросетевая модель

-Librosa — извлечение музыкальных признаков из аудио

-SQLite — легковесная база данных

HTML/CSS/JavaScript — фронтенд (без фреймворков)

Установка зависимостей вручную
Если вы не используете start.bat, зависимости устанавливаются командой:

```bash
pip install fastapi uvicorn torch torchaudio transformers librosa scipy numpy
```
